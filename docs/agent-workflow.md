# Agent Workflow

AgentProof is designed to document AI agent-assisted builds.

## Build Loop

1. Define project goal and PRD.
2. Generate app structure and implementation tasks.
3. Build the local web app.
4. Validate with local commands.
5. Capture evidence: screenshots, terminal output, repo links, generated artifacts.
6. Generate a proof pack for reviewers.

## Evidence Types

- Terminal logs
- Agent session summaries
- GitHub repository links
- Live demo URLs
- Screenshots or recordings
- Generated Markdown artifacts
- Billing or usage screenshots
