# AgentProof Xiaomi MiMo Proof Pack

## Executive Summary
AgentProof is a reviewer-ready proof-pack generator for AI agent builders. It helps developers applying to grants, hackathons, and model credit programs turn raw evidence into structured, human-reviewable dossiers.

## Public Links
- GitHub: https://github.com/HakimIqbal/agentproof
- Live demo: https://agentproof.vercel.app

## AI Agent Workflow
- Primary tools: Hermes Agent and Codex
- Model series: routed models / MiMo-compatible workflow
- Stages: PRD -> scaffold -> build -> verify -> deploy -> proof pack

## Claims and Evidence
1. Project exists and is open source -> GitHub repository
2. Product is usable without login -> Vercel demo
3. AI agents were used in build workflow -> terminal/session screenshots
4. Output is reviewer-ready -> generated Markdown proof pack

## Impact
AgentProof reduces manual application prep time and makes AI-assisted work easier to audit.
